# 🛠️ 一、 我对 Butterfly 主题做了哪些“魔改”？

Butterfly 主题虽然好看，但默认样式太“素”了。为了让它更符合我的气质，我主要动了以下几个地方：

### 1. 基础门面装修
修改了**站点配置文件** `_config.yml` 和 **主题配置文件** `butterfly/_config.yml`：
- **头像 & 作者名**：换成了stewie的帅照和我的ID。
- **公告栏**：写了一段极客范儿的自我介绍。
- **背景图**：换成了温馨的 "best-family.jpg"。

### 2. 样式透明化 (高级感来源) ✨
为了让背景图能透出来，我加了一段自定义 CSS 代码。（不知道怎么没搞出来）
文件路径：`source/css/custom.css````css

/* 让首页文章卡片变透明 */
#recent-posts > .recent-post-item {
    background: rgba(255, 255, 255, 0.6) !important;
    backdrop-filter: blur(10px);
}
/* 让侧边栏小卡片变透明 */
#aside-content .card-widget {
    background: rgba(255, 255, 255, 0.6) !important;
    backdrop-filter: blur(10px);
}

# 二、 以后怎么发新帖子？

这是给自己看的备忘录，防止以后忘了命令。

### 第一步：新建文章
在博客根目录下打开黑窗口（终端），输入：`hexo new "文章标题"`
系统会自动在 source/_posts 目录下生成一个 .md 文件。

### 第二步：编辑“帽子” (Front-matter)
打开生成的 md 文件，最上面的部分非常重要，决定了文章长啥样;
可以参考这篇post

`aside: false`   <-- 单篇文章加上这行，拒绝侧边栏
启用了侧边小图标

### 第三步：正文排版技巧
标题：用 `#`号，两个` ##`是二级标题，三个`###`是三级。

加粗：用` **文字**`。

引用：用` > 文字`.就是"`       `"

图片：`![](图片链接)`。

# 🚀 三、 预览与发布
写完之后，必须执行“三连击”才能看到效果：`hexo clean && hexo g && hexo s`
然后打开浏览器访问 `http://localhost:4000`欣赏杰作！😎
 
1、清理缓存（防止样式不更新）：`hexo clean`

2、生成网页： `hexo g`

3、本地预览： `hexo s`

满意用.bat文件发布
